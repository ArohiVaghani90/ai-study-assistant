# Personal Study Assistant

A simple AI-powered study assistant web app that helps students with explanations, practice problems, and summaries.

---

## Features

- Ask questions in plain English
- Get step-by-step explanations
- Helpful for math, science, and general study topics
- Clean and simple user interface

---

## Tech Stack

- Next.js
- TypeScript
- React
- OpenAI API

---

## Running Locally

```bash
npm install
npm run dev
